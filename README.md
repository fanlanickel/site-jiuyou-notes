# site-jiuyou-notes

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容相互独立，不针对任何特定网站或域名，仅作为静态页面集合进行管理与展示。

## 项目简介

本项目主要目的是将零散的 HTML 页面集中归档，方便后续查阅、分享或部署。所有页面均为纯前端静态文件，不依赖后端服务，可直接在浏览器中打开运行。

## 目录说明

```
site-jiuyou-notes/
├── index.html          # 仓库入口页面（如有）
├── pages/              # 存放各独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── assets/             # 公共资源文件（如样式、图片、脚本等）
    ├── css/
    ├── js/
    └── images/
```

- **pages/**：存放所有独立 HTML 页面，每个文件代表一个独立的归档页面。
- **assets/**：存放页面共用的静态资源，如样式表、脚本文件或图片。

## 页面归档说明

- 每个 HTML 页面均为独立完整的文档，可直接通过浏览器访问。
- 页面内容不涉及具体网站、域名或商业推广。
- 归档页面可能包含文字、图片、表格等常见 HTML 元素，用于记录或展示信息。
- 页面命名尽量简洁直观，便于识别。

## 维护说明

- 本仓库仅用于归档已有 HTML 页面，不主动更新或修改页面内容。
- 如需新增页面，请将文件放入 `pages/` 目录，并确保文件路径正确。
- 若页面依赖外部资源（如 CDN 链接），请确认其可用性。
- 欢迎提交 Issue 或 Pull Request 进行页面补充或问题反馈。
